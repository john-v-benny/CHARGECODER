### Prerequisites

```bash
# System Requirements
- Python 3.8+
- Node.js 16+
- Git
```

### Installation

```bash
# Clone repository
git clone https://github.com/ASHISH-28-02/CHARGECODER.git && cd CHARGECODER

# Backend setup
cd Back-end
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate

# Frontend setup
cd ../Front-end
npm install
npm start
```
